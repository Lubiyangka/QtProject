## 二 游戏介绍

### 2.1 操作

方向键“上”，“下”，“左”，“右”

| 按键   | 操作          | 备注                                  |
| ------ | ------------- | ------------------------------------- |
| ↑      | 向上移动      | 默认初始方向为向上移动                |
| ↓      | 向下移动      |                                       |
| ←      | 向左移动      |                                       |
| →      | 向右移动      |                                       |
| 空格键 | 长按加速      |                                       |
| ESC键  | 开始/暂停游戏 | 初始状态为暂停状态，按下ESC键开始游戏 |
