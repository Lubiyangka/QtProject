## 二 游戏介绍🎉

### 2.1 操作

方向键“上”，“下”，“左”，“右”

| 按键  | 操作                   |
| ----- | ---------------------- |
| W(8)  | 向上移动               |
| S(2)  | 向下移动               |
| A(4)  | 向左移动               |
| D(6)  | 向右移动               |
| J     | 长按加速（仅单人模式） |
| ESC键 | 开始/暂停游戏          |

> 默认初始方向为向上移动，括号为双人模式玩家二的操作按钮

> 初始状态为暂停状态，按下ESC键开始游戏
